# Monte Carlo Simulation Based Portfolio Risk and Return Forecasting Framework

This project performs portfolio risk and return forecasting using Monte Carlo simulation.

## Project Overview

The project analyzes selected Indian stocks and builds different portfolio strategies:

- Equal-weight portfolio
- Mean-variance optimized portfolio
- Minimum volatility portfolio

It also performs:

- Historical price analysis
- Daily log return calculation
- Portfolio risk-return comparison
- Monte Carlo simulation
- VaR and CVaR calculation
- Stress testing
- Sensitivity analysis
- Regression analysis against NIFTY 50

## Stocks Used

- RELIANCE.NS
- TCS.NS
- HDFCBANK.NS
- SUNPHARMA.NS
- MARUTI.NS
